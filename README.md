# 🎓 Student Analytics Dashboard

An interactive Student Analytics Dashboard built using Python, Streamlit, PostgreSQL and Plotly.

## Features

- Interactive dashboard
- PostgreSQL database integration
- Student performance analysis
- Course-wise statistics
- City-wise analytics
- Dynamic filtering
- CSV download
- Database statistics
- Modern responsive UI

## Technologies Used

- Python
- Streamlit
- PostgreSQL
- Plotly
- Pandas
- Psycopg2

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
