# 🎓 Student Analytics Dashboard

An interactive Student Analytics Dashboard built using **Python**, **Streamlit**, **PostgreSQL**, and **Plotly** to analyze student data through dynamic visualizations and filters.

---

## 📌 Project Overview

This dashboard provides insights into student performance, enrollment, placements, attendance, and other academic metrics. It allows users to interactively explore data using filters and visual charts.

---

## ✨ Features

- 📊 Interactive KPI Cards
- 📈 Dynamic Plotly Charts
- 🔍 Sidebar Filters
- 🗃️ PostgreSQL Database Integration
- 📥 Download Filtered Data as CSV
- 📋 Sortable Student Records Table
- 📊 Database Statistics Section
- 🕒 Last Updated Timestamp
- 🎨 Clean & Responsive UI
- ⚡ Fast Performance using Streamlit

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/AARSHVASHISHT/Student-Analytics-Dashboard.git
```

Move into the project folder

```bash
cd Student-Analytics-Dashboard
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
